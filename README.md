# 💰 Financial Dashboard

A modern full-featured financial dashboard built with Next.js, TypeScript, and Firebase.  
Track your income, expenses, and savings in a clean and interactive interface.

🔗 Live Demo: https://financial-dashboard-1ef1e.web.app

---

## ✨ Features

- 📊 Cashflow analytics (income vs expenses)
- 💸 Transactions management (add, edit, delete)
- 🏦 Savings goals with progress tracking
- 📈 Charts and data visualization (Chart.js)
- 🔐 Authentication (Firebase Auth: Email + Google)
- 🌙 Light / Dark mode
- 💱 Multi-currency support
- 📱 Fully responsive UI
- ⚡ Optimized performance with modern React patterns

---

## 🛠 Tech Stack

**Frontend**
- Next.js (App Router)
- React
- TypeScript
- Tailwind CSS

**State Management**
- Zustand

**Backend / Services**
- Firebase Authentication
- Firebase Firestore

**Charts**
- Chart.js + react-chartjs-2
